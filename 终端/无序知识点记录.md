# Mac命令行

### 查看所有运行端口号：
```bash
lsof -i -P | grep LISTEN
```

### 下载所有运行端口号：
```bash
lsof -i -P | grep -i listen > ports.txt
```

### 杀死某个端口号，先查看所有运行端口号，获取PID
```bash
kill 6424
```

### 杀死某类端口号，先查看所有运行端口号
```bash
killall node
```
